# Java To-Do List App

A simple console-based To-Do List application built with Java. This project allows users to add, display, update, and remove tasks using a command-line interface.

## Features

- Add new tasks
- Show all tasks
- Update existing tasks
- Remove tasks
- Simple menu-based console interaction
- Input validation for task selection

## Technologies Used

- Java
- ArrayList
- Scanner
- Switch Case
- Loops and Conditional Statements

## Project Structure

The application includes the following functionalities:

1. **Add Task** – Add a new task to the list
2. **Show Tasks** – Display all saved tasks
3. **Remove Task** – Delete a task by index
4. **Update Task** – Edit an existing task
5. **Exit** – Close the application

## How to Run

1. Clone the repository:

```bash
git clone <your-repository-link>
```

2. Open the project in your Java IDE (IntelliJ IDEA, Eclipse, VS Code, etc.)

3. Compile and run the `Main.java` file.
